# zen-Babel for macOS 0.8.2 修正リスト

## 不具合修正

- 空白のみセグメントが翻訳対象として登録される問題を修正
  - 文間の空行や改行のみが単独セグメント化された場合に、翻訳不能のオレンジマーカーが出ることがありました。
  - 文字を含まないセグメントは生成しないように修正しました。
  - 既存の改行/空白レイアウトは joiner 側で保持し、本文の再構成互換性を維持しています。

- macOS バンドル名/成果物名の表記を `zen-Babel` に統一
  - `.app` / `.zip` / `.dmg` の命名を `zen-Babel` ベースに揃えました。
  - 実行バイナリ名・Bundle ID・URL スキーム（`prebabellens://`）は互換性のため維持しています。
  - Issue: #2

## 補足

- 翻訳処理は引き続きオンデバイスで実行されます。
