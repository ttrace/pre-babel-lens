# Pre-Babel Lens v0.6.0 (Draft)

## User-facing update / ユーザー向けアップデート

This release makes translation output feel more natural and readable, especially for long passages with paragraph breaks.
このリリースでは、特に段落を含む長文で、翻訳結果の読みやすさと自然さを改善しました。

## Highlights

- Faster initial output
  - The beginning of the translation appears sooner, so long text feels more responsive.
  - 翻訳の先頭部分がより早く表示されるようになり、長文でも待ち時間が短く感じられます。
- Better paragraph preservation
  - Paragraph breaks are kept more reliably, so adjacent sections are less likely to run together.
  - 段落の切れ目がより安定して保持され、前後の文章がつながってしまう問題が起きにくくなりました。
- Handling Apple Intelligence limitations
  - When Apple Intelligence cannot translate part of a text, the app inserts the original text in a different color.
  - Apple Intelligence の制限のために翻訳できない部分を、色違いの原文で挿入します。

## Notes

- This release targets Apple Intelligence-enabled environments.
- このリリースは Apple Intelligence が利用可能な環境を対象としています。
