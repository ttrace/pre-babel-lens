# Pre-Babel Lens v0.5.0 (Draft)

## Small but fun update

DeepL-like quick launch is here.
Copy the same text twice (`Cmd+C`, `Cmd+C`) and Pre-Babel Lens pops to front and starts translation.

## Highlights

- Added quick launch by double copy (`Command + C`, `Command + C`) on macOS
- Improved app front activation when launched from double copy
- Improved streaming UI responsiveness by extracting partial `translation` from `snapshot.rawContent.jsonString`
- Added a guard for invalid outputs where `translation` becomes only a `kind` label

## Notes

- This release targets Apple Intelligence-enabled environments.
- Foundation Models availability and model asset readiness are still required at runtime.

## Verification

- Automated tests: `swift test` (passed)
- Manual checks (recommended):
  - double-copy launch from another app
  - app comes to front reliably
  - translated body text appears normally (not only kind labels)
